# Application / Agent Runtime Issues

This directory contains atomic, report-ready security issues specific to the Application / Agent Runtime trust boundary.

## What Belongs Here

Individual security issues that target application integration and agent behavior:
- Direct and indirect prompt injection
- Tool misuse and privilege escalation
- Agent hijacking and loss of control
- MCP/plugin vulnerabilities
- Business logic manipulation

Each issue page should be self-contained and follow the canonical template.

## How to Create a New Issue

1. Copy the structure from the Model issues template
2. Replace all placeholder text with specific content
3. Ensure all 11 required sections are present
4. Use framework IDs only (no prose) in the Framework References section
